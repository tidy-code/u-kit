Universal UI-kit

ui-kit is a simple ui elements set created with WebComponents

Getting Started

To start using u-kit elements, you can link it like so (preferably in the <head> section of the page):

 <script src="#link" defer ></script>

Elements:


Modal

It's a simple modal with 2 slots:

  • Main slot (no name)
  • header slot

And if you set bind attribute, it will automatically bing close events to close the modal. Otherwise you need to handle it by yourself (with "change" event)

<u-modal id="modal" bind>
  <h4 slot="header">Do you really want to do this?</h1>
  <p class="mb-5">Lorem ipsum dolar sit amet</p>
  <footer class="pt-3 text-right">
    <button class="btn btn-outline-secondary mr-2" onclick="closeModal()">Cancel</button>
    <button class="btn btn-success" onclick="closeModal()">Yes!</button>
  </footer>
</u-modal>
